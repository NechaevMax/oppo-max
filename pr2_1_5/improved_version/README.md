# Улучшенная версия (ПР-2.1.5)

Что изменено по сравнению с исходной версией:
- логика разбора строк и работы с файлами выделена в отдельные модули (Parser/Repository/Manager);
- добавлены собственные исключения и валидация входных данных;
- исправлены риски UB/ошибок ввода (без `cin >>` в бесконечном цикле без очистки);
- добавлены модульные тесты (без внешних зависимостей);
- добавлен скрипт `tools/docgen.py` для автогенерации Markdown-документации.

## Сборка

```bash
mkdir build && cd build
cmake ..
cmake --build .
ctest
```
