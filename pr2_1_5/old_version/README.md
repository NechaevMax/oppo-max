# Исходная версия (ПР-3)

Консольная программа «Меню блюд»:
- чтение данных из файла `menu.txt` (строки вида `menu name="..." price=... time=...`);
- вывод всех блюд;
- сортировка по времени приготовления;
- фильтрация по времени;
- добавление блюда вручную.

**Примечание:** исходный код содержит `#include <windows.h>` для установки кодировки консоли в Windows.
Для сборки в Linux/macOS в папке `include/` добавлена минимальная заглушка `windows.h`.
